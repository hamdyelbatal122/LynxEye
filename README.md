# Responsive Stairs Diagram

A **pure CSS responsive stairs diagram** — a visual UI component that renders a stepped/staircase layout using only HTML and CSS. Clean, lightweight, and fully responsive.

## ✨ Features

- 🎨 Built with pure HTML & CSS — no JavaScript required
- 📱 Fully responsive across all screen sizes
- ⚡ Lightweight with zero dependencies
- 🔧 Easy to customize (colors, sizes, labels)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/hamdyelbatal122/Responsive-Stairs.git
   cd Responsive-Stairs
   ```

2. **Open in browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` in your file explorer.

## 🎨 Customization

You can easily customize the stairs by modifying the CSS variables:

```css
/* Example: Change step color */
.step {
  background-color: #4A90E2;
  height: 40px;
}
```
